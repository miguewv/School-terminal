# Simple School Terminal

This is a simple Python project I made to practice programming.
I made it in about one hour.

The program simulates a small school terminal where you can:

- register students
- view registered students
- add grades
- view a student's grades
- check the total number of students

It is a basic project created for learning and improving my Python skills.
